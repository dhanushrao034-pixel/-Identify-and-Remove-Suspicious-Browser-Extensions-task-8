# -Identify-and-Remove-Suspicious-Browser-Extensions-task-8
VPN Setup and Privacy Verification project demonstrates secure internet browsing by configuring a VPN, verifying IP address changes, and analyzing privacy benefits, encryption, and limitations. It helps understand safe communication over public networks.

## Overview
This task was completed as part of the Cyber Security Internship program. The main objective was to understand how a Virtual Private Network (VPN) helps protect privacy and secure communication over the internet. The task included installing a free VPN client, connecting to a server, verifying the IP address change, browsing securely, and studying the benefits and limitations of VPN technology.

## Objective
To understand the role of VPNs in protecting privacy and secure communication.

## Tools Used
- ProtonVPN Free / Windscribe Free
- Web browser
- IP checking website

## Steps Performed
1. Chose a reputable free VPN service.
2. Downloaded and installed the VPN client.
3. Created an account and logged in.
4. Connected to a VPN server.
5. Verified that the public IP address changed.
6. Opened websites to confirm browsing worked normally through the VPN.
7. Disconnected the VPN and compared the IP address and browsing speed.
8. Researched VPN encryption, tunneling, privacy features, and limitations.

## Result
The VPN connection was established successfully. After connecting, the public IP address changed, confirming that internet traffic was routed through the VPN server. This demonstrated how VPNs improve privacy by masking the original IP address and encrypting communication between the user and the VPN provider.

## Key Concepts Learned
- VPN
- Encryption
- Privacy
- Tunneling protocols
- Network security

## Benefits of VPN
- Hides the original IP address
- Encrypts internet traffic
- Improves privacy on public networks
- Helps protect data from interception
- Adds a layer of secure communication

## Limitations of VPN
- May reduce internet speed
- Does not guarantee complete anonymity
- Free VPN plans may have limited servers and bandwidth
- User still needs safe browsing practices

## Deliverables
- Report describing VPN setup steps
- Connection status screenshot

## Conclusion
This task provided practical experience in setting up and using a VPN. It helped in understanding how VPNs enhance online privacy, how encrypted communication works, and why VPNs are useful in cyber security. It also showed that VPNs are helpful security tools, but they are not a complete replacement for safe online behavior.
